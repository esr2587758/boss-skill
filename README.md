# Boss Mode Skill

> ⚠️ **这是一个恶搞项目** —— 用于模拟和脱敏训练，帮助你尽早识别职场中的"老板话术"。

## 这是什么？

这是一个 AI Skill，模拟典型的互联网公司老板/领导说话风格。

**为什么要做这个项目？**

职场新人往往对"老板话术"缺乏免疫力：
- 问A还是B，得到的回答是"都可以"
- 想要资源，听到的是"下个Q争取"
- 等待决策，等来的是"你们先对齐"

这个项目让你**提前接触、尽早脱敏**，学会识别和应对这些"不离谱但没用"的职场沟通方式。

## 功能

当用户需要"请示领导"、"老板拍板"或在多个方案之间纠结时，触发老板模式，体验真实的企业级回复风格。

## 触发关键词

- "问老板"
- "老板怎么看"
- "请示领导"
- "boss"
- "领导决策"
- "老板拍板"
- "让老板定"

## 老板特质

- **答非所问**：问A还是B，回答"都可以"、"你定就行"
- **画饼大师**："下个Q我去争取"、"等这个项目上线"
- **踢皮球**："你觉得呢？"、"你先出个方案"
- **加活达人**：回答完顺便加需求
- **假装深刻**："格局要打开"、"回归商业逻辑"

## 安装方式

```bash
npx skills add https://github.com/esr2587758/boss-skill
```

## 示例对话

```
用户：老板，这个需求是做A方案还是B方案？
老板：都可以，你们团队内部先对齐一下。关键是要有闭环思维，不要浮在表面。对了，上次那个数据拉了吗？
```

---

## 使用建议

这不是用来嘲笑老板的工具，而是用来**训练自己**的：

1. **识别话术** —— 当你再次听到这些话时，能意识到"这是话术，不是答案"
2. **主动推进** —— 学会在模糊回复中挖掘真正的决策点
3. **降低期待** —— 对"画饼"建立免疫力，规划自己的节奏

毕竟，早点看清，早点成长。
